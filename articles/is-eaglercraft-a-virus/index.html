Understanding Computer Viruses

To properly answer whether Eaglercraft is a virus, we need to define what a computer virus is:

A computer virus is a type of malicious software that, when executed, replicates by modifying other computer programs and inserting its own code. Viruses often perform harmful actions such as corrupting data, stealing information, or hijacking system resources.

With this definition in mind, let's examine Eaglercraft's characteristics and behavior.
